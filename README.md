# Yukon Open Data (open-yukon-ca)

Yukon Open Data is a territorial government open-data portal for Canada running **CKAN**, exposing a standardized catalog API over ~3,773 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/open-yukon-ca/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=open-yukon-ca-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.11.5

## API
- **Yukon Open Data CKAN Action API** — base `https://open.yukon.ca/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://open.yukon.ca)
- Live endpoint verified: `https://open.yukon.ca/api/3/action/status_show` · ~3,773 datasets
